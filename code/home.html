About Me

Hi, my name is Ishita Agrawal, a business analyst with a background in Computer Science and Engineering.

I work on data, getting meaningful insights out of data is something I really like. Data visualization and Machine Learning are some techniques that I use to achieve my goal using different BI tools and technologies.
